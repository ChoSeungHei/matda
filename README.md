# Matda (나만의 맛집 다이어리)
위치 기반 맛집 리뷰 등록 시스템

## 개발 환경
FrontEnd : React, CSS

BackEnd : GCP, Nginx, Nodejs

라이브러리 : react-bootstrap, react-geocode, react-router-dom, react-rating-stars-component, react-icons, google-maps-react

## 개발 데모

1. 전체 화면 구성
    - 현재 위치 지도
    - 현재 위치 지역랭킹 Top 3
    - 나만의 랭킹 Top 3
